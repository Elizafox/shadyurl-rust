ShadyURL-Rust
-------------
This is like [ShadyURL](https://github.com/Elizafox/ShadyURL) but written in Rust and better done.

Copy the env_example file to .env, edit it, run `sea-orm-cli migrate -d <database URL>`, and you're off to the races.
